# 💾 SQL Portafolio

Bienvenido a mi portafolio profesional de SQL.

Este repositorio reúne los ejercicios, prácticas y proyectos desarrollados durante mi formación en Administración de Bases de Datos y mi Maestría en Ciencia de Datos.

El objetivo de este portafolio es demostrar mis habilidades en diseño, consulta, administración y optimización de bases de datos relacionales.

---

# 🚀 Tecnologías

- SQL
- MySQL
- PostgreSQL
- DBeaver
- pgAdmin
- Git
- GitHub
