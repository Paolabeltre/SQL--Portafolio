# 💈 Proyecto SQL: Estilo Urbano Salón & Barbería

## 📌 Descripción

Este proyecto consiste en la construcción de una base de datos relacional para la empresa **Estilo Urbano Salón & Barbería**.

La base de datos permite administrar clientes, empleados, categorías de servicios, servicios, productos, citas y los servicios asociados a cada cita.

---

## 🎯 Objetivos

- Crear una base de datos relacional.
- Diseñar tablas con claves primarias y foráneas.
- Insertar datos realistas.
- Realizar consultas SQL básicas e intermedias.
- Aplicar `ORDER BY`, funciones de agregación, `JOIN`, `UPDATE` y `DELETE`.

---

## 🛠 Tecnologías utilizadas

- SQL Server
- T-SQL
- SQL Server Management Studio
- GitHub

---

## 📂 Tablas creadas

- Clientes
- Empleados
- Categorias_Servicio
- Servicios
- Productos
- Citas
- Cita_Servicio

---

## 🔎 Consultas realizadas

1. Consulta con `SELECT` y `ORDER BY`.
2. Consulta con funciones de agregación.
3. Consulta con `JOIN` entre tablas.
4. Actualización con `UPDATE` y condición `WHERE`.
5. Eliminación con `DELETE` y condición `WHERE`.

---

## 🧠 Aprendizaje

Con este proyecto reforcé el diseño de bases de datos relacionales, la creación de relaciones entre tablas y el uso de consultas SQL para obtener información útil de un negocio real.
